## Why Law Assistant?

**Law Assistant** is a professional-grade legal consulting platform built on top of **LangChain** and the **Model Context Protocol (MCP)**. It is designed to provide accurate, context-aware answers to complex legal questions by leveraging specialized search engines and state-of-the-art LLMs.

<CardGrid stagger>
	<Card title="MCP Tool Integration" icon="setting">
		Native support for **Model Context Protocol**. Seamlessly connects to `legal-mcp` for Japanese laws and `webmcp` for real-time web context.
	</Card>
	<Card title="Legal Intelligence Engine" icon="magnifier">
		Built-in deep search capabilities for legal documents using Elasticsearch and advanced retrieval patterns.
	</Card>
	<Card title="Unified LLM API" icon="random">
		Switch between **OpenAI**, **Anthropic**, **ollama** or any provider like **OpenRouter**, with a single configuration change.
	</Card>
	<Card title="Multi-Interface" icon="laptop">
		Access via a modern **React Web UI**, a powerful **CLI**, or integrated directly into **LINE Bot**.
	</Card>
</CardGrid>
