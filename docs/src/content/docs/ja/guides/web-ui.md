title: ウェブ UI ガイド
description: 法律アシスタントのウェブチャットインターフェースの使用方法。
---

ウェブ UI は、ユーザーが法律アシスタントと対話するための主要な方法です。複数のセッションと詳細な AI 回答をサポートする、洗練されたモダンなチャットインターフェースを提供します。

## はじめる

1.  アプリケーションを起動し、`http://localhost:3000` にアクセスします。
2.  認証情報でログインします。
    - デフォルト管理者: `admin@example.com` / `Admin123!@#`

## 主要機能

### 1. チャットダッシュボード
メイン画面には、法律に関する質問を送信できるクリーンなチャットエリアがあります。回答は AI が処理するにつれて、1 文字ずつストリーミングされます。

### 2. セッションサイドバー
左側にはチャット履歴が表示されます。以下の操作が可能です：
- **新しいセッションの作成**: 新しい会話を開始します。
- **セッションの切り替え**: 以前の法的相談に戻ります。
- **セッションの削除**: 履歴を整理します。

### 3. モデル選択
チャット設定では、そのセッションで使用する LLM を選択できます (例: GPT-4o, Claude 3.5 Sonnet)。ただし、設定で有効になっているモデルに限ります。

## UI でのツールの使用

ツールを手動でトリガーする必要はありません。例えば「労働契約法の解雇規定について教えてください」と尋ねると、チャットエリアに「法的ドキュメントを検索中」や「ウェブで確認中」といったステータスインジケーターが表示されます。

## ユーザー設定

- **プロファイル**: パスワードの変更やアカウント詳細の確認ができます。
- **ログアウト**: セッションを安全に終了します。

## トラブルシューティング

- **接続が切れた場合**: チャットが応答しなくなった場合は、ページをリフレッシュしてください。システムはデータベースからチャット履歴を自動的に復元します。
- **認証エラー**: セッションが期限切れになった可能性があります。再度ログインして、新しい JWT トークンを取得してください。
