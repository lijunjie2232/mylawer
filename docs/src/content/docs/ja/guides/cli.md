CLI (Command Line Interface) は、開発者、自動化スクリプト、およびブラウザを使わずに素早くクエリを実行したい場合に最適です。

## 基本的な使い方

### インタラクティブモード

複数の質問ができる永続的なセッションを開始します：

```bash
pnpm cli
```

起動したら、質問を入力して Enter キーを押すだけです。終了するには `exit` と入力するか、`Ctrl+C` を押します。

### 単一クエリモード

引数として直接質問を渡します：

```bash
pnpm cli "日本における残業代のルールは何ですか？"
```

## 高度な機能

### パイプとリダイレクト

他のコマンドやファイルから質問をパイプすることができます：

```bash
# テキストファイルから質問する
cat question.txt | pnpm cli

# 回答をファイルに保存する
pnpm cli "民法第1条を要約してください" > summary.txt
```

### JSON 出力

自動化のために、構造化されたデータが必要な場合があります：

```bash
pnpm cli "クエリ" --json
```

## CLI の設定

CLI はメインサーバーと同じ `.env` 設定を使用します。`LLM_API_KEY` と `DATABASE_URL` が設定されていることを確認してください。

また、CLI も `ALLOWED_MCP_TOOLS` を尊重します。CLI セッションで特定のツールを無効にしたい場合は、別の `.env.cli` ファイルを使用できます：

```bash
# 特定の環境ファイルで CLI を実行
DOTENV_CONFIG_PATH=.env.cli pnpm cli
```

## ユースケース

- **CI/CD 統合**: ドキュメント内の法的コンプライアンスを自動的にチェック。
- **ローカルリサーチ**: 法律 ID やテキストの素早い検索。
- **データスクレイピング**: 複数の法的シナリオを一括処理し、結果を保存。
