## なぜ法律アシスタントなのか？

**法律アシスタント (Law Assistant)** は、**LangChain** と **Model Context Protocol (MCP)** を基盤としたプロフェッショナルな法的コンサルティングプラットフォームです。専門的な検索エンジンと最新の LLM を活用することで、複雑な法的質問に対して文脈に応じた正確な回答を提供できるように設計されています。

<CardGrid stagger>
	<Card title="MCP ツールの統合" icon="setting">
		**Model Context Protocol** をネイティブサポート。日本の法律を検索する `legal-mcp` や、リアルタイムのウェブ情報を取得する `webmcp` にシームレスに接続します。
	</Card>
	<Card title="法的インテリジェンスエンジン" icon="magnifier">
		Elasticsearch と高度な検索パターンを使用した、法的ドキュメントの組み込み深層検索機能。
	</Card>
	<Card title="統一 LLM API" icon="random">
		設定を一つ変更するだけで、**OpenAI**、**Anthropic**、または **OpenRouter** 経由のあらゆるプロバイダーを切り替え可能。
	</Card>
	<Card title="マルチインターフェース" icon="laptop">
		モダンな **React ウェブ UI**、強力な **CLI**、または **LINE Bot** 経由で直接アクセス。
	</Card>
</CardGrid>
